# Factura-AI 🚀

Sistema inteligente de automatización contable que utiliza **n8n**, **Python (uv)** y **Gemini 1.5 Flash** para el procesamiento automático de facturas.

## 🌟 Características
- **Extracción Multimodal:** Procesa facturas en formato PDF, JPG y PNG con alta precisión.
- **IA de Vanguardia:** Implementa Gemini 1.5 Flash para extraer más de 30 campos específicos (IVA, Netos, CUIT, etc.).
- **Flujo n8n:** Automatización de extremo a extremo: Google Drive -> IA -> Google Sheets.
- **Estructura Empresarial:** Organización diaria automática de archivos y logs.
- **CI/CD Ready:** Preparado para despliegues automáticos a múltiples clientes mediante GitHub Actions.

## 🛠️ Stack Tecnológico
- **Workflow Engine:** n8n (Server-side)
- **AI Model:** Google Gemini 1.5 Flash
- **Language:** Python 3.13+ managed by `uv`
- **CI/CD:** GitHub Actions
- **Cloud Storage:** Google Drive & Sheets

## 🚀 Vibe Coding Journey
Este proyecto fue desarrollado utilizando técnicas de *Vibe Coding*, optimizando el ciclo de desarrollo mediante el uso estratégico de asistentes de IA para la arquitectura de sistemas, generación de scripts de extracción y orquestación de flujos de trabajo.

## 🔒 Seguridad
Las credenciales y claves de API se gestionan estrictamente a través de variables de entorno y secretos de GitHub, garantizando un despliegue seguro para cada cliente.
